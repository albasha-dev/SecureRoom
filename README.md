# SecureRoom - منصة التواصل المشفر

منصة تواصل مشفرة من طرف إلى طرف (E2EE) تعتمد على غرف محادثة مؤقتة أو دائمة مع قاعدة بيانات SQLite.

## المميزات الجديدة:
- **قاعدة بيانات SQLite**: يتم حفظ الغرف والرسائل (المشفرة) والعمليات في قاعدة بيانات محلية.
- **Backend Node.js**: تم تحويل المشروع من تطبيق client-side فقط إلى تطبيق full-stack.
- **Socket.io**: دعم التواصل الحقيقي بين مختلف المتصفحات والأجهزة.
- **تشفير E2EE**: يتم تشفير الرسائل في المتصفح قبل إرسالها وفك تشفيرها عند المستلم فقط.

## متطلبات التشغيل:
- تثبيت [Node.js](https://nodejs.org/).

## طريقة التشغيل:

1. افتح المجلد في Terminal.
2. قم بتثبيت المكتبات المطلوبة:
   ```bash
   npm install
   ```
3. ابدأ تشغيل الخادم:
   ```bash
   npm start
   ```
4. افتح المتصفح على العنوان التالي:
   ```
   http://localhost:3000
   ```

## هيكل المشروع:
- `server.js`: الخادم الرئيسي ومنطق قاعدة البيانات.
- `public/index.html`: واجهة المستخدم والمنطق البرمجي للمتصفح (التشفير).
- `database.db`: ملف قاعدة بيانات SQLite (يتم إنشاؤه تلقائياً).
- `package.json`: ملف تعريف المشروع والمكتبات.
